Bem vindo à API de teste para Iterup

Esta API está usando o token de segurança Microsoft.AspNetCore.Authentication.JwtBearer.

Para acessar algumas rotas basta você cadastrar uma pessoa, e fazer o login. O sistema retornará o token para você.

Se você estiver utilizando o Postman, na aba Authorization, selecione o TYPE "Bearer Token" e cole-o no campo "Token"

As UFs estão sendo carregadas na aplicação, utilize a rota Get /api/pessoas/ufs para visualizar os Ids.

Rotas Ação: Exige Autenticação
Post /api/pessoas/?cpf&senha? Rota para autenticação Não
Get /api/pessoas Consulta a lista de objetos Pessoas Não
Get /api/pessoas/{id} Consulta objeto Pessoa pelo código Sim
Get /api/pessoas/uf/{uf} Consulta objeto Pessoa pelo código da UF Sim
Post /api/pessoas/{nome,cpf,datanascimento,iduf,senha} Adiciona uma Pessoa Não
Put /api/pessoas/{id}{nome,cpf,datanascimento,iduf,senha} Atualiza uma Pessoa pelo seu Id Sim
Delete /api/pessoas/{id} Exclui uma pessoa pelo seu Id Sim
